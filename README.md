# LittleJS TS Webpack Starter (New+ directory)

To get started with this project:

- [X] Rename project in `package.json`
- [X] Change project name in `index.html` title tag
- [X] Run `npm i` in terminal to install all the dependencies
- [ ] (Optional) Run `git init` to initialize a GitHub repository
- [ ] Run `npm run dev` to build then start the development server
